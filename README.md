# Rayat Khan - iOS Developer Portfolio

A responsive portfolio website for Rayat Khan, an iOS Developer with 4+ years of experience building production iOS and tvOS applications.

## Live Portfolio

Portfolio link:

```text
https://rayatkhan690.github.io/ios-journey-portfolio/
```

Add this link anywhere you want to share the portfolio, including LinkedIn, WhatsApp, email signatures, CV, and GitHub profile overview.

If the link shows `404`, enable GitHub Pages:

1. Open this repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Select `GitHub Actions` as the Pages source.
4. Save and wait a few minutes for deployment to finish.
