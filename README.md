# iOS Journey Portfolio

A responsive iOS developer portfolio built with HTML, CSS, and JavaScript.

## Sections

- Hero introduction
- Featured iOS projects
- Skills panel
- iOS journey timeline
- Contact links
- Dark and light theme toggle

## Customize

Update these placeholders before publishing:

- Name and title in `index.html`
- Project GitHub links
- Email address
- GitHub profile URL
- LinkedIn profile URL

## Run Locally

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 5173
```

Then visit:

```text
http://127.0.0.1:5173
```
